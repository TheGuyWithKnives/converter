# 3D Studio Pro - New Features

Aplikace byla kompletně přepracována na profesionální 3D editor s pokročilými funkcemi.

## Nové funkce

### 1. Tab-based Layout
- **Upload & Generate tab** - Pro nahrávání a generování 3D modelů
- **3D Viewer tab** - Profesionální viewport s pokročilými nástroji
- Automatické přepnutí na viewer po vygenerování modelu

### 2. Enhanced 3D Viewer
Zcela nový viewer s pokročilými funkcemi:
- Sidebar panel s nástroji po pravé straně
- Real-time preview všech změn
- Moderní tmavý UI design

### 3. Material Editor
Pokročilý editor materiálů:
- **Base Color** - Nastavení barvy materiálu
- **Metalness** - Kovový efekt (0 = nekovový, 1 = plně kovový)
- **Roughness** - Drsnost povrchu (0 = lesklý, 1 = matný)
- **Emissive** - Světelná emise z povrchu
- **Material Presets** - Připravené materiály (Plastic, Metal, Glass, Gold, Wood, Stone)

### 4. Mesh Statistics
Kompletní statistiky modelu:
- Počet trojúhelníků
- Počet vertexů
- Počet meshů a materiálů
- Rozměry modelu (X, Y, Z)
- Kvalita poly (Low/Medium/High)

### 5. Camera Presets
Rychlé přepínání pohledů:
- Front, Back, Left, Right
- Top, Bottom
- Isometric view
- Perspective view

### 6. Viewport Controls
Ovládání zobrazení:
- **Show Grid** - Zobrazení mřížky
- **Wireframe Mode** - Drátový model
- **Show Lights** - Zobrazení světel
- **Bounding Box** - Ohraničující box
- **Reset Camera** - Reset kamery do výchozí polohy

## Vylepšení designu

- Moderní profesionální UI v tmavém režimu
- Gradientové akcenty a ikony
- Responzivní layout
- Lepší kontrast a čitelnost
- Intuitivní ovládání

## Technické vylepšení

- Nová komponenta `EnhancedGLBViewer` s integrovanými nástroji
- Modulární architektura s oddělenými komponentami
- Lepší správa stavu
- Optimalizovaný rendering

## Jak používat nové funkce

1. **Nahrajte obrázek** v Upload & Generate tabu
2. **Vygenerujte 3D model** pomocí AI nebo Basic módu
3. Automaticky se otevře **3D Viewer tab**
4. **Upravte materiál** pomocí Material Editor
5. **Změňte úhel pohledu** pomocí Camera Presets
6. **Přepněte zobrazení** pomocí Viewport Controls
7. **Sledujte statistiky** v Mesh Statistics panelu

## Kompatibilita

Všechny původní funkce zůstávají zachovány:
- Image to 3D konverze (AI i Basic mode)
- Multi-angle support
- Image editor
- Print estimator
- Model animator
- Export do různých formátů (GLB, OBJ, STL, PLY)
